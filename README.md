# VuePressTemplate-Reco-v1

![GitHub](https://img.shields.io/github/license/shadowfish07/VuePressTemplate-Reco-v1)

快速使用 VuePress！本模板集成了 VuePress 1.x 和 VuePress 主题 reco 1.x，可以在它的基础上自由修改。

## 使用方法

首先安装依赖

```
npm install
```

开启本地 dev 服务器

```
npm run docs:dev
```

输出用于部署的静态文件

```
npm run docs:build
```
